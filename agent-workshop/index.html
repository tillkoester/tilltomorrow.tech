08 / Questions

Questions

Do I need to be technical?

You don't need to be a developer. But you should be comfortable following a setup process step by step. If you've ever installed an app or configured a tool and made it work, you'll be fine.

What do I need before the first session?

A computer, an internet connection, and one task you genuinely want off your plate.

What does it cost to keep the agent running after the workshop?

Around €6–10 per month. No software subscriptions, no per-seat fees, no lock-in.

What tool are we actually building with?

Hermes — an open-source AI agent built by Nous Research. Currently the most-used open-source agent in the world, with persistent memory, self-improving skills, and support for any AI model you choose. It's free, and you own it completely.

Why not just use the AI already built into my existing tools?

Those work for single tasks in isolation. They don't connect across your work, don't remember you, and you're tied to whoever provides them. What we build here learns how you work — across everything — and keeps getting better.

Will I be able to maintain it myself after the workshop?

Yes. That's part of what the three sessions are for.
