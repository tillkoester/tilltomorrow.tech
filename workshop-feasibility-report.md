# KI-Automation Workshop — Feasibility Report

## Executive Summary
Ein 3x3h Workshop **ist machbar** für Nicht-Techniker. Zentrale Anpassung: Docker muss durch Cloud-Environments/VPS ersetzt werden.

## Key Findings
1. **Zielgruppe:** Solopreneure, Kleingewerbe, Freelancer ohne Dev-Background
2. **Format:** 3 Sessions à 3h = absolutes Minimum
3. **Technik:** Vorkonfigurierte Cloud-VMs statt Docker
4. **Tools:** n8n (low-code), Make.com, OpenAI API
5. **Pre-Work:** Account-Erstellung (30min), keine lokale Installation

## Empfohlener Plan
| Session | Inhalt |
|---------|--------|
| Pre-Work | Accounts + Video-Einführung |
| Session 1 | Basics & Erster Workflow (3h) |
| Session 2 | Eigene Automation bauen (3h) |
| Session 3 | Deployment & Ausblick (3h) |
| Post-Work | 14-Tage Challenge + Follow-up |
